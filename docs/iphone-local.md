# Uso da iPhone sulla rete locale

1. Avviare il server di sviluppo:

```sh
npm run dev
```

2. Trovare l'IP del Mac:

```sh
ipconfig getifaddr en0
```

3. Aprire da iPhone:

```text
http://IP_DEL_MAC:3000
```

4. Verificare che Mac e iPhone siano sulla stessa rete Wi-Fi.

5. Se serve, disattivare temporaneamente il firewall del Mac.
